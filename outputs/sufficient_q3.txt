A) Encoding is the process of putting information into memory so that it can be stored and later retrieved. It basically means taking information and saving it in the memory system.

B) The testing effect is when taking practice tests or quizzes helps you do better on later exams. Being tested lets you see what you know and what you don’t, so you can study those areas and often perform better on the final test.

C) Working memory is a limited-capacity system that holds information in mind while you are actively using it, such as when you are solving a problem, doing mental arithmetic, or following directions. It involves keeping information temporarily available and mentally manipulating it. Without rehearsal, information in working memory typically lasts for about 15 to 20 seconds before it fades.

D) Implicit memory is the kind of memory people use when they intentionally recall facts and events from their past. Explicit memory is the type of memory involved in skills and habits, like riding a bike or tying your shoes, which you don’t really think about.
